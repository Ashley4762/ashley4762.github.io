Interactive Computer Graphics

HTML5 Canvas • JavaScript • Computer Graphics

Project Overview

This project demonstrates an interactive computer graphics application developed using HTML5 Canvas and JavaScript. It focuses on raster graphics algorithms, sprite rendering, transformations, and real-time keyboard and mouse interaction.

Project Features

  • 🎨 Pixel Drawing
  • 📏 Line Drawing
  • ⬜ Rectangle
  • 🟦 Filled Rectangle
  • 🌈 Gradient Rectangle
  • ⭕ Ellipse Drawing
  • 🖼 Sprite Rendering
  • 🔄 Rotation Transformation
  • 🔍 Scale Transformation
  • 🎮 Keyboard Interaction
  • 🖱 Mouse Interaction
  • 🌼 24-bit Sprite Rendering

Keyboard Controls

Key Action
1 Change Color to Red
2 Change Color to Green
3 Change Color to Blue
← Rotate Left
→ Rotate Right
↑ Scale Up
↓ Scale Down

Mouse Interaction

  • Click and drag to draw.
  • Move the mouse to interact with graphics.
  • Supports real-time interaction.

Technologies

  • HTML5
  • CSS3
  • JavaScript (ES6)
  • HTML5 Canvas API
