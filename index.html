Interactive Computer Graphics

HTML5 Canvas • JavaScript • Raster Graphics

Project Overview

This project demonstrates an interactive raster graphics application developed using HTML5 Canvas and JavaScript. Users can draw graphics and manipulate sprites through keyboard and mouse interaction.

Project Features

  • ✔ Pixel Drawing
  • ✔ Draw Line
  • ✔ Draw Rectangle
  • ✔ Filled Rectangle
  • ✔ Gradient Rectangle
  • ✔ Draw Ellipse
  • ✔ Sprite Rendering
  • ✔ Sprite Rotation
  • ✔ Sprite Scaling
  • ✔ 24-bit Sprite
  • ✔ Mouse Interaction
  • ✔ Keyboard Interaction

Keyboard Controls

Key Function
1 Red Color
2 Green Color
3 Blue Color
← Rotate Left
→ Rotate Right
↑ Scale Up
↓ Scale Down

Mouse Interaction

• Click and drag to draw on the canvas.

• Move the mouse to change sprite position.
